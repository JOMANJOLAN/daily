# 线程和进程

* 进程：独立地址空间、独立资源
* 线程：进程内部的执行流，共享同一份地址空间
  * 共享：代码、数据、文件描述符、堆
  * 私有：栈、寄存器、线程 ID、信号掩码、errno

# pthread

```bash
gcc main.c -o main -pthread
```

## 创建线程

```c
#include <pthread.h>

int pthread_create(
    pthread_t *thread,
    const pthread_attr_t *attr,
    void *(*start_routine)(void *),
    void *arg
);
```

* thread：输出线程 ID
* attr：属性（NULL 用默认）
* start_routine：线程入口函数
* arg：传给线程的参数

## 等待线程结束（阻塞）

```c
#include <pthread.h>

int pthread_join(pthread_t thread, void **retval);
```

* 类似进程的 wait()，主线程阻塞等待子线程退出。

## 线程自己退出

```c
#include <pthread.h>

void pthread_exit(void *retval);
```

* 不退出进程，只退出当前线程

## 获取自身id

```c
#include <pthread.h>

pthread_t pthread_self(void);
```

* Linux下pthread_t的实现是一个整数，类似pid，但是与pid是相互独立的，因为pthread是库，不属于内核，而内核不应该依赖用户态，用户态不应该绑架内核

## 分离线程（Detached）

```c
#include <pthread.h>

// 方式1：创建后设置
pthread_detach(pthread_self());

// 方式2：用属性创建（推荐）
pthread_attr_t attr;
pthread_attr_init(&attr);
pthread_attr_setdetachstate(&attr, PTHREAD_CREATE_DETACHED);
pthread_create(&tid, &attr, func, NULL);
pthread_attr_destroy(&attr);
```

## 线程栈大小

```c
#include <pthread.h>

pthread_attr_t attr;
pthread_attr_init(&attr);
pthread_attr_setstacksize(&attr, 1024*1024*16); // 16MB
pthread_create(&tid, &attr, func, arg);
```

# 互斥锁（Mutex）：解决竞争

## 定义与初始化

```c
#include <pthread.h>

pthread_mutex_t mutex;
// 静态初始化（推荐）
pthread_mutex_t mutex = PTHREAD_MUTEX_INITIALIZER;
// 动态初始化
pthread_mutex_init(&mutex, NULL);
```

## 加锁 / 解锁

```c
#include <pthread.h>

// 加锁（阻塞）
int pthread_mutex_lock(pthread_mutex_t *mutex);
// 解锁
int pthread_mutex_unlock(pthread_mutex_t *mutex);
// 销毁
int pthread_mutex_destroy(pthread_mutex_t *mutex);
```

# 条件变量（Condition Variable）：等待 / 唤醒

## 定义初始化

```c
#include <pthread.h>

pthread_cond_t cond;
// 静态
pthread_cond_t cond = PTHREAD_COND_INITIALIZER;
// 动态
pthread_cond_init(&cond, NULL);
```

## 等待

```c
#include <pthread.h>

int pthread_cond_wait(
    pthread_cond_t *cond,
    pthread_mutex_t *mutex
);
```

## 唤醒

```c
#include <pthread.h>

// 唤醒一个
int pthread_cond_signal(pthread_cond_t *cond);
// 唤醒所有
int pthread_cond_broadcast(pthread_cond_t *cond);
```

# 自旋锁（spinlock）

拿不到锁不睡眠，原地循环自旋

```c
#include <pthread.h>

pthread_spinlock_t lock;
pthread_spin_init(&lock, 0);
pthread_spin_lock(&lock);
pthread_spin_unlock(&lock);
```

# 读写锁（rwlock）最实用的锁

```c
#include <pthread.h>

pthread_rwlock_t rwlock;
pthread_rwlock_init(&rwlock, NULL);

pthread_rwlock_rdlock(&rwlock);   // 读锁
pthread_rwlock_wrlock(&rwlock);   // 写锁
pthread_rwlock_unlock(&rwlock);
```

# 信号量（semaphore）

```c
#include <semaphore.h>

sem_wait(&sem); // P操作：-1，不够就等
sem_post(&sem); // V操作：+1，唤醒等待者
```
