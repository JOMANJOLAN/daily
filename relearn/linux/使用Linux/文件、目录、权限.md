# ls -l

```
-rwxr-xr-- 1 user group 1234 Mar 18 10:00 test.sh
```

1. 文件类型：-
2. 所有者权限：rwx
3. 所属组权限：r-x
4. 其他用户权限：r--

# 权限

## 文件

```
r  可读（cat、less）
w  可写（修改、保存）
x  可执行（运行程序 / 脚本）
```

## 目录

```
r  能列出目录内容（ls）
w  能创建 / 删除 / 重命名文件
x  能进入目录（cd），能访问里面的文件
```

## 数字表示

```
r = 4
w = 2
x = 1
```

## 常用权限

```
644  文件默认（所有者可写，其他只读）
755  目录 / 可执行文件默认
777  全开（不推荐）
```

# chmod改权限

## 调用chmod的条件

* 对/bin/chmod有x权限
* 是目标文件的所有者或者用户是root

## 字母方式

```bash
chmod u+x file  # 给所有者加执行
chmod g-w file  # 去掉组的写权限
chmod o+r file  # 给其他人加读
chmod a+r file  # 所有用户加读
```

## 数字方式

```bash
chmod 644 file
chmod 755 dir
```

---

# chown 改所有者/组

```bash
chown user file        # 改所有者
chown user:group file  # 改所有者和组
chown -R user dir      # 递归改目录下所有
```

---

# 特殊权限

## SUID（4xxx）

* 只对可执行文件生效
* 运行时临时拥有文件所有者权限

```bash
chmod 4755  # passwd
```

## SGID (2xxx)

* 文件：运行时继承组权限
* 目录：新建文件继承目录的组

```bash
chmod 2755 dir
```

## Sticky 粘滞位 (1xxx)

* 只对目录生效
* 目录内文件只能自己删自己的

```bash
chmod 1777 dir
```

---

# 默认权限

* 文件默认：644 = 666 & ~umask
* 目录默认：755 = 777 & ~umask
* 普通用户umask通常是002
* root umask通常是022
* umask只在创建的时候起作用
* 编译后的可运行程序是编译器手动加了x权限

# 命令行

```bash
ls          # 看文件
ls -l       # 详细信息
ls -la      # 看隐藏文件
cd dir      # 进入目录
cd ..       # 返回上一级
pwd         # 看当前路径
mkdir dir   # 新建文件夹
rm file     # 删除文件
rm -r dir   # 删除文件夹
cp src dst  # 复制
mv src dst  # 移动 / 重命名
```

```bash
cat file    # 直接看全部
less file   # 分页看（上下翻页）
head file   # 看开头
tail file   # 看结尾
tail -f     # 实时看日志
```

```bash
chmod 755 file         # 修改权限
chown user:group file  # 修改所有者
id                     # 看自己 UID/GID
groups                 # 看自己在哪些组
```

```bash
# 除了add，其余命令user、group使用数字、文本均可
useradd user  # 新建用户
passwd user   # 设置密码
usermod -g group user   # 修改主组
usermod -aG group user  # 加入附加组（最常用）
gpasswd -d user group   # 从组删除
groupadd group  # 新建组
```

```bash
ps aux       # 看所有进程
top          # 实时看系统负载
kill PID     # 杀进程
sudo 命令    # 以 root 运行
```

```bash
find 路径 -name 文件名  # 找文件
grep '内容' 文件        # 在文件里搜文字
```
